Real-time aggregate metrics from Cassandra proxy instances worldwide. All data is anonymous — only token counts, compression ratios, and cost estimates are collected. Telemetry is off by default; [opt in](https://github.com/Sarthak-47/cassandra/blob/main/cassandra/telemetry/beacon.py) with `CASSANDRA_TELEMETRY=on`.
